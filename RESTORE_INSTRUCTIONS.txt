EMAIL-SAFE COPY OF JSing

Some email systems block JavaScript/TypeScript files inside zip attachments.
In this archive, code files with blocked extensions were renamed by appending .email-safe.txt.

After unzipping, restore them by running this from inside the unzipped JSing folder in PowerShell:

Get-ChildItem -Recurse -File -Filter "*.email-safe.txt" | Rename-Item -NewName { $_.Name -replace "\.email-safe\.txt$", "" }

Then install dependencies and run the app:

cd web
npm install
npm run dev
