# NYC Job Search Tracker

Use this file as the single source of truth for current pipeline status.

## Status Flow

`Found -> Scored -> Applied -> Outreach Sent -> Screen -> Interview -> Offer/Closed`

## Compensation Target

- Primary target: roles with base salary starting at `180K-200K+`.
- Default filter: skip roles with posted base max below `180K` unless there is a strong referral or exceptional upside.
- Record posted compensation range in tracker for every scored role.

## 100-Point Scoring Rubric

Score each role before applying.

| Category | Max Points | Scoring Guidance |
| --- | ---: | --- |
| NYC-based or NYC hybrid relevance | 20 | 20 = clear NYC onsite/hybrid, 10 = remote with NYC preference, 0 = non-NYC |
| Retail/commerce/enterprise workflow relevance | 15 | 15 = strong retail/workflow domain match, 8 = partial match, 0 = weak/no match |
| AI, product, transformation, or systems modernization angle | 15 | 15 = core scope, 8 = supporting scope, 0 = absent |
| Seniority fit (7-12 years, manager/senior manager equivalent) | 10 | 10 = direct fit, 5 = slight stretch, 0 = major mismatch |
| Keyword alignment with profile strengths | 15 | Award points based on count and quality of strong keyword matches |
| Cross-functional business + technical ownership | 10 | 10 = clear bridge role, 5 = mostly one-sided, 0 = no bridge |
| Leadership/accountability scope | 10 | 10 = visible ownership and decision authority, 5 = execution-only, 0 = minimal scope |
| Brand, pay, network, or career-option value | 5 | 5 = high strategic value, 3 = moderate, 0 = low |

## Priority Thresholds

- `75+`: Apply now.
- `60-74`: Apply selectively if capacity allows.
- `<60`: Skip unless strong networking/referral wedge.

## Scoring Notes

- Use one sentence in `Notes` to justify the final score.
- If uncertain between two lanes, pick the lane that matches top 3 JD responsibilities.
- Re-score roles when JD updates or referral signals appear.
- If compensation is not posted, mark as `Unknown` and prioritize recruiter clarification before applying.
