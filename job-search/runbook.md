# NYC Job Search Runbook

This runbook defines the recurring weekly operating loop and completion criteria.

## Service Levels

- Outreach sent within 24 hours of application.
- Follow-up sent 5 business days after no response.
- Weekly review completed every Friday.
- Compensation filter enforced: prioritize roles with posted base starting at `180K-200K+`.

## Weekly Timeboxes

- Monday (60-90 min): Source and score roles, prepare first application packet batch.
- Tuesday (45-60 min): Send recruiter and hiring manager outreach.
- Wednesday (45-60 min): Networking and referral requests.
- Thursday (60-90 min): Second application packet batch and submissions.
- Friday (45-60 min): Follow-ups, metrics review, and strategy updates.

## Weekly Execution Cadence

1. Monday
   - Add 20-30 NYC roles to intake.
   - Filter out roles with posted base max below `180K` unless there is a strong exception case.
   - Score all roles and mark priority tier.
   - Prepare packets for all `75+` roles.
   - Submit first batch (target 5-8).
2. Tuesday
   - Send recruiter and hiring-manager outreach for Monday applications.
   - Request at least 3 warm referrals.
3. Wednesday
   - Networking block: 8-10 targeted messages tied to active applications.
4. Thursday
   - Prepare and submit second batch (target 5-8).
   - Complete outreach for Thursday submissions.
5. Friday
   - Send follow-ups on stale applications.
   - Complete weekly review and define next-week adjustments.

## Conversion Metrics

- Response rate = `responses / applications`.
- Screen rate = `screens / applications`.
- Interview rate = `interviews / applications`.
- Outreach coverage = `(recruiter outreach + hiring manager outreach) / applications`.
- Lane conversion = `lane screens / lane applications`.

## Optimization Rules

- If response rate is below 10% for 2 weeks, narrow applications to `80+` score roles and tighten bullet tailoring.
- If one lane underperforms by 50%+ versus others for 2 weeks, reduce that lane's weekly share and reallocate effort.
- If outreach coverage is below 90%, reduce application volume next week until SLA is recovered.
- If a title family generates zero screens after 8+ applications, rotate title variants and update search queries.
- If a company cluster shows strong signal (responses/screens), increase weekly focus on similar companies.
- If salary-fit rate is below 70% of sourced roles, adjust query terms to include compensation-focused keywords and prioritize transparent-pay employers.
