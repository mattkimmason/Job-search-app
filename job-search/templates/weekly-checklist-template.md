# Weekly Checklist Template

## Core Weekly Tasks

- [ ] Source new NYC-only roles.
- [ ] Score every role with the rubric.
- [ ] Build packets for all 75+ score roles.
- [ ] Submit first batch (Mon) and second batch (Thu).
- [ ] Send recruiter outreach within 24 hours.
- [ ] Send hiring manager outreach within 24 hours.
- [ ] Follow up on roles with no response after 5 business days.
- [ ] Complete Friday weekly review and update strategy.

## Weekly Exit Criteria

- [ ] Response, screen, interview, and outreach coverage rates are computed.
- [ ] Lowest-performing lane/title segment has one explicit adjustment.
- [ ] Search query library is updated with at least one improvement.
